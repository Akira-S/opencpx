Scott, when you get a chance, how does this sound:

 

 

Disabling a virtual host domain

Disabling a virtual host domain temporarily disables web and mail services
on the domain. While disabled, users will not be able to retrieve mail or
access the account through the Control Panel, FTP or shell (telnet or ssh).
Domain settings and mail are not deleted when the domain is disabled. 

 

Deleting a virtual host domain

Deleting a virtual host domain permanently removes all of the virtual host
information from the server. The system cannot delete a virtual host domain
until all End Users associated with the domain are deleted.
